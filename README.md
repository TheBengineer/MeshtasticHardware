# [Meshtastic Open Hardware List](https://meshhardware.thebengineer.net/)

This [website](https://meshhardware.thebengineer.net/) is a compiled list of [Meshtastic](https://meshtastic.org/) and [Meshcore](https://meshcore.io/) 
compatible hardware for the Mesh community. 

Contact me if you want to add your hardware to the list, or if you have any questions.
